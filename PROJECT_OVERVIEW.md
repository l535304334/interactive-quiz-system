# 项目概述 — 老大专用刷题系统

> 主文件：`[题库]交互式刷题页面.html`（约 2.7MB 单文件 HTML 应用）
> 版本：v1.0.0 · 状态：已发布，进入维护模式

---

## 一句话定位

一个纯前端、零依赖后端、数据全部本地化的交互式刷题系统，通过 SM-2 间隔重复算法、5 种答题模式与可视化学习报告，帮助考生高效复习与冲刺。

---

## 项目背景与目标

刷题是应试复习的核心环节，但传统纸质题库存在三大痛点：

1. **错题管理低效**：错题本靠手抄，无法快速回看与统计
2. **复习节奏失控**：不知道哪些题该今天复习、哪些可以延后
3. **学习数据丢失**：纸笔记录易丢，云题库又涉及账号、联网、隐私

本项目的目标是用一个**单文件 HTML 应用**解决上述问题：

- 双击即开即用，无需安装、无需注册、无需联网
- 全部数据存于浏览器 `localStorage`，可随时导出 JSON 备份
- 内置 SM-2 间隔重复算法，自动安排今日复习清单
- 提供 5 种答题模式 + 复合模式，适配不同复习场景
- Chart.js 可视化学习报告，量化复习进度

## 适用人群

- **医学考试备考者**（默认题库类别为：体格检查 / 共选 / 实验室检查 / 问诊诊断 / 心电图）
- 任何需要**高频刷题 + 错题追踪 + 间隔复习**的应试考生
  - 通过 JSON 导入可承载任意学科题库（考研政治/英语/数学、408 计算机、执业资格等均可）
- 注重数据隐私、不愿把学习记录交给云服务的用户
- 偏好单文件、可离线、零依赖工具的极简主义者

> 说明：系统本身与学科无关，默认 `DEFAULT_SOURCES` 是医学方向，但通过"类别管理"和"JSON 导入"可承载任意题库。

## 主要特性概览

### 答题与导航
- 5 种答题模式：错题 / 复习（SRS）/ 收藏 / 浏览 / 考试
- 复合模式：任意叠加 2 种以上（如"错题+复习"），统一聚合到当前题目流
- 顺序 / 随机洗牌切换
- 类别筛选下拉（动态读取题库中的所有 source）
- 错题次数筛选（1+/2+/3+/5+/10+ 次错误）
- 上一题 / 下一题 / 重做当前题 / 撤销最近一次作答（Ctrl+Z）

### 题库管理
- JSON 导入 / 导出（含全部进度与 SRS 状态）
- 自定义题（`nextCustomId` 自增，本地新增）
- 类别管理（动态增删改）：空类别可删，重命名保证引用完整性
- 自动快照系统：每次关键写操作触发 `triggerAutoBackup()`，最多 10 份
- 启动时自动检测最新快照，提示恢复（`checkAutoRestore`）
- localStorage 配额监控（10MB 警告阈值）

### 学习辅助
- **SM-2 间隔重复**：`updateSRS(qid, correct)` 根据对错更新 interval/repetition/easiness
- 今日复习清单（`getTodayReviewCount`）：按 `nextDate <= today` 计算
- 计时系统：今日时长 `todayStudySeconds` + 累计时长 `studySeconds` + 距目标剩余
- 自定义闹钟：可设分钟数 + 循环开关，到点提示休息
- 每日目标：`dailyGoal` 题数 + 进度环 + 连续打卡天数 `streakDays`
- 学习报告：Chart.js 4 图表 + 学科正确率排行 + 错题 Top 榜

### 数据安全
- 5 类自动快照触发点：题目变更、错题变更、已答变更、收藏变更、错次变更
- 快照索引 `snapIndex` + 快照实体 `snap_<ts>`，超限自动淘汰最旧
- 旧版 `autoBackupData` 兼容回退
- 导入数据格式校验（`try/catch` + 字段补全）
- `escapeHtml()` 全覆盖防止 XSS

### 交互体验
- 键盘模式（`K` 切换）：A-I 选选项、1-9 备选、←→ 切题、Space 下一题、`/` 搜索
- 全局快捷键：W 错题 / V 复习 / F 收藏 / B 浏览 / E 考试 / R 报告 / G 目标 / T 计时 / H 帮助 / M 自定义题
- 跟随系统暗色模式（`prefers-color-scheme` + `body.dark` 类切换）
- 11 个模态弹窗全部带 `role="dialog"` `aria-modal="true"`
- `prefers-reduced-motion` 降级动画
- 触摸滑动切题（移动端友好）

### 视觉设计
- 玻璃态 + 蓝紫渐变（header `backdrop-filter: blur(20px)`）
- 4 组胶囊导航布局（`nav-group`）
- 3 层阴影体系（`--sh-1/2/3`）
- cubic-bezier 动效（`--ease` / `--ease-bounce`）
- 左侧色条装饰（`--c-bar-1` / `--c-bar-danger`）
- 完整 CSS Custom Properties 设计 Token（`--c-*` / `--r-*` / `--sh-*` / `--fs-*` / `--sp-*` / `--t-*` / `--ease-*`）
