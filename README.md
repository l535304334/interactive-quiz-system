# 老大专用刷题系统

> 一个纯前端、零依赖、双击即用的本地刷题系统。基于 SM-2 间隔重复算法、Chart.js 数据可视化与玻璃态视觉系统，所有数据存于浏览器 localStorage，不上传、不联网、不泄露。

内置临床医学题库样例（共 5944 题，覆盖体格检查、实验室检查、问诊诊断、共选、心电图 5 个分类），可按需通过 JSON 导入替换为任意学科题库。

---

## 项目亮点

1. **零安装、零依赖、双击即用** — 单文件 HTML 应用，无构建工具、无后端、无数据库，双击即在浏览器打开。
2. **SM-2 间隔重复算法** — 基于遗忘曲线的智能复习调度，答对拉长间隔（1→6→更久），答错次日复习，按钮实时显示今日待复习题数。
3. **5 种答题模式任意组合** — 错题 / 复习 / 收藏 / 浏览 / 考试，支持 2 种以上模式同时开启形成复合模式（交集刷题）。
4. **Chart.js 学习报告** — 4 张图表（30 天答题量折线、正确率趋势、各科雷达、错题分布 doughnut）+ 顽固错题 Top 8 排名 + 最强/最弱科目建议。
5. **完整数据安全闭环** — 自动快照（最多 10 份，含 SRS 进度）+ 启动时备份检测恢复 + JSON 导入导出 + 合并/覆盖两种导入策略 + 存储配额预警。
6. **XSS 防护全覆盖** — 所有动态 HTML 插入均经 `escapeHtml()` 转义；JSON 导入 `try/catch` 校验；纯前端无网络请求。
7. **键盘流全快捷操作** — A-I / 1-9 选选项，←→ 切题，Space 下一题，Backspace 重做，Ctrl+Z 撤销，W/F/X/B/V/E/R/K/G/T/H/M 单键切换模式，`/` 聚焦搜索。
8. **可访问性与降级** — ARIA `role="dialog"` `aria-modal="true"` 全模态覆盖、Esc 关闭、点击遮罩关闭、`prefers-reduced-motion` 自动降级动效、跟随系统暗色模式。

## 功能展示

### 答题核心

- 点击选项即时判对错，答对 0.5s 自动跳转，答错停留
- 错题本：自动收录 + 单题移除 + 一键清空 + 错次徽章（错 N 次标记）
- 收藏夹：⭐ 标记重点题，考前速记专属
- 错题分类筛选：全部 / 1 次+ / 2 次+ / 3 次+ / 5 次+ / 10 次+
- 顺序 / 随机 / 未做优先 三种刷题模式 + 科目筛选

### 浏览速记模式

- 全部 / 自定义数量 / 科目范围 三种浏览方式
- 与错题/复习/收藏自由组合
- 每页 20 / 50 / 100 / 200 题可选分页

### 模拟考试

- 选题源（支持"错题本"组卷）
- 题量：10 / 20 / 30 / 50 / 100 五档
- 单题限时（秒）+ 整场限时（分钟）双重计时
- 分页作答（20 题/页），交卷自动判分 + 错题自动收录

### 学习报告

- 近 30 天答题量折线图
- 正确率趋势折线图
- 各科完成度 + 正确率雷达图
- 错题分布 doughnut 图
- 顽固错题 Top 8 排名（带奖牌）
- 最强 / 最弱科目 + 智能学习建议

### 间隔复习（SM-2）

- 答错 → 1 天后复习
- 答对 → 间隔拉长（1 → 6 → round(interval × ef)）
- 难度因子 ef ∈ [1.3, 2.5+]，答错降低、答对抬升
- 按钮实时显示今日待复习题数

### 全局搜索

- 关键词搜题干 / 选项 / 答案
- 输入 `#ID` 按题号精准定位（如 `#123`）
- 高亮匹配片段
- 键盘 ↑↓ Enter 跳转，`/` 快捷键聚焦
- 分批构建索引，不阻塞主线程

### 计时与闹钟

- 今日时长 + 累计时长 + 距目标 三项指标
- 自定义闹钟：小时 + 分钟，可选循环 / 仅一次
- 一键预设：5 分 / 25 分 / 45 分 / 90 分
- 与"1 小时起每 30 分钟"系统休息提醒以先到者响
- 弹窗内"再 5 分钟"延迟

### 数据管理

- 题库：批量增删 + JSON 导入导出
- 类别管理：新增 / 重命名 / 删除空类别
- 备份：多版本快照（最多 10 份，含 SRS 复习进度）
- 合并导入：不清空，只追加不重复的题
- 重置进度：可视化选择，一键清空
- 存储占用检测：超过 10MB 主动提示导出

### 跨端体验

- 跟随系统主题（亮 / 暗自动切换）
- 每日目标 + 答题计时
- 手机滑动切题 + 移动端"更多"菜单
- 答题动画 + 鼓励系统 + 模态缩放弹入

## 使用说明

### 基本操作流程

1. **首次打开** — 顶部导航栏显示 4 组胶囊按钮：
   - 辅助组：科目下拉 / 顺序切换 / 键盘模式
   - 模式组：错题 / 复习 / 收藏
   - 工具组：浏览 / 考试
   - 信息组：介绍 / 指南（移动端折叠进"更多 ▼"）

2. **答题** — 点击题目下方选项即判对错；答对自动跳下一题，答错停留可查看解析。

3. **切换模式** — 点击任一模式按钮即开启该模式，再次点击关闭。多模式同时开启时进入复合模式（取交集）。

4. **搜索** — 点击"更多 ▼ → 搜索"或按 `/` 键打开搜索框；输入 `#123` 可直接跳到第 124 题（id=123）。

5. **查看报告** — 点击"更多 ▼ → 报告"打开学习报告，查看 4 张图表与学习建议。

6. **备份 / 导出** — 点击"更多 ▼ → 数据"打开数据管理弹窗，可手动快照、导出 JSON、合并导入、类别管理。

7. **设置目标 & 闹钟** — 点击进度条上的目标条打开目标设置；点击底部 ⏱ 按钮打开计时与闹钟弹窗。

### 数据安全建议

- **定期导出** — localStorage 可被浏览器清空，建议每周通过"数据 → 导出"下载 JSON 备份
- **多版本快照** — 系统自动保留最近 10 份快照，可在"数据"弹窗查看时间线并一键恢复
- **合并导入** — 换设备时选择"合并导入"，可保留原进度并追加新题
